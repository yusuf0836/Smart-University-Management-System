REST API documentation for the Smart University Management System built with Laravel 12.

    <strong>Base URL</strong>: <code>http://localhost</code>
</aside>

# Welcome

Welcome to the **Smart University Management System API** documentation.

This API provides endpoints for managing:

- Authentication
- Faculties
- Departments
- Teachers
- Students
- Semesters
- Courses
- Enrollments
- Results
- Attendance
- Notices
- Fees
- Examinations
- Dashboard
- Transcripts

All endpoints return JSON responses and require authentication unless otherwise specified.
